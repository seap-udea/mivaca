# Novedades

## v.2026-01-17

- Creación de la página Novedades para ver los últimos cambios
- La app ahora es bilingüe (ES/EN), oh yeah!
- Se agregaron características avanzadas opcionales:
  - Fusionar cuentas de comensales
  - Repartir un producto entre comensales seleccionados (no necesariamente entre todos)
- El vaquero ahora puede confirmar el pago de un comensal (por ejemplo, si se le descargó el celular)
- Se puede configurar el porcentaje de la propina (por defecto 10%)
